## Features

- **Natural Conversation:** Interact with SUR using speech recognition for hands-free chatting.
- **Weather Updates:** Fetch real-time weather information for any city.
- **News Updates:** Get the latest general and technology news instantly.
- **Multimedia Control:** Play music, open camera, and browse websites directly via voice commands.
- **Chat History:** All conversations are saved locally for future reference.
- **AI-Powered Responses:** Uses OpenRouter AI to provide smart, context-aware replies.
